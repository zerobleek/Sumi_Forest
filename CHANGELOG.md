# Changelog

All notable changes to Sumi Forest will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-02

### Added
- Initial release of Sumi Forest
- Two-tab interface: Todo List and Forest View
- Todo list with CRUD operations
- Quick add functionality for rapid task creation
- Swipe actions for complete, edit, and delete
- Task editor with title, note, due date, and priority
- Filter by All, Today, Upcoming, and Completed
- Sort by Created, Due Date, and Priority
- Forest visualization with watercolor trees
- Four tree species based on priority: Bamboo, Maple, Pine, Sakura
- Tree growth animation
- Poisson-disk sampling for natural tree layout
- Tree detail view showing task information
- SwiftData persistence
- Haptic feedback for interactions
- Dark mode support
- Full accessibility with VoiceOver labels
- Dynamic Type support
- Reduce Motion support
- Comprehensive error handling
- Categorized logging with os.Logger
- Unit tests for models and layout
- Preview support for all views
- Sample data generator
- Comprehensive documentation

### Technical Details
- iOS 17.0+ requirement
- Swift 5.9+
- SwiftUI + SwiftData
- MVVM architecture
- No external dependencies
- Zero warnings build
- All tests passing
